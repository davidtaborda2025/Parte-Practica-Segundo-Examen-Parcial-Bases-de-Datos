En este repositorio se encuentra el trabajo desarrollado por el estudiante David Taborda Montenegro, del curso de Bases de Datos.

Para entender la estructura del proyecto, el cual fue llevado a cabo usando herramientas de software como IntelliJ IDEA, Docker Desktop y PostgreSQL, se deben tener en cuenta estas aclaraciones:

1. En la única carpeta del proyecto, llamada "Parte Practica P2", reposan otra carpeta y dos archivos. La subcarpeta (llamada "database") contiene a los archivos DDL (schema) y DML (inputs) del trabajo, al igual que un Dockerfile para
   montar la base de datos.

2. Los otros dos archivos corresponden a la vista gráfica del modelo relacional (el diagrama relacional) dado para esta resolución, el cual fue hecho en Markdown dentro de IntelliJ. Y, el segundo archivo, corresponde a las consultas hechas
   durante el desarrollo, con el objetivo de preservarlas. Dicho archivo es queries.sql .

Con ello, se tiene todo el desarrollo completo a esta segunda parte práctica para los exámenes parciales del curso.
